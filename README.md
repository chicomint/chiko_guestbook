<h1>How to use?</h1>
<p>Clone this git and go to the file then run : "bun install" and  "bun add @elysiajs/html"</p>
<p>to deploy that using: "bun src/index.ts"</p>
<p>look at "localhost:8080" </p>
<p>(≧◡≦) ♡</p>


<h3>Configuration</h3>
<p>You can configure your MongoDB connection in <code>config.json</code>:</p>
<pre><code>{
  "mongoUri": "mongodb://127.0.0.1:27017/guestbook",
  "collectionName": "messages"
}</code></pre>
